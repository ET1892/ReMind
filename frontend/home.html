ReMind

What is ReMind?
ReMind is my final year project! It is a website dedicated to human memory education. I have taken the approach of 'gamifying' memory education.
Create an Account
You can create an account by clicking here. When you create an account, your game scores will be saved.
Learn
If you want to learn more about memory, how it is, created, compounded and lost, click here.
Games
This website is about making cognitive exercise fun. I have curated five games which have been shown to improve cognition and prevent cognitive decline. You can view your game history by clicking the 'Game History' button after signing in.
Facts
If you want to learn more, visit the facts page, where you can discover a new fact with the click of a button!
Light & Dark Mode
To switch themes, click the moon icon in the top right corner for dark mode, or the sun icon to switch to light mode."
